Welcome to Grimlock

The Password Cracker Game!

Welcome to Grimlock! This interactive game challenges you to crack passwords of increasing difficulty using logic, pattern recognition, and cybersecurity skills. Each level introduces new password types and real-world attack scenarios, helping you learn why strong passwords and good security habits matter.

  • Tutorial: Learn the basics by unscrambling a simple password. Practice using the workings box and see how hints work before starting the real challenge.
  • Level 1: Crack a very weak password (like a simple word or number). See how easily attackers can guess common passwords.
  • Level 2: Unscramble a weak or medium password (may include numbers or uppercase letters). Learn how small changes add some security, but are still vulnerable.
  • Level 3: Unscramble a strong password (includes uppercase, lowercase, numbers, and symbols). Discover why complexity makes passwords harder to crack.
  • Level 4: Combine words and numbers to find the password. See how attackers use wordlists and patterns to break these.
  • Level 5: Spot and solve repeating or keyboard patterns. Many people use these, but so do hackers!
  • Level 6: Break a simple cipher. Learn how even "encrypted" passwords can be vulnerable if the cipher is weak.
  • Level 7: Phishing simulation! Test your ability to spot a phishing email and avoid social engineering attacks.
  • Level 8: Learn about password managers and why they are essential for staying secure online.

Use the workings box under each input to jot down your ideas or try combinations. If you get stuck, use the Clear Hint button (at a points cost) for help. You can Give Up at any time to end the game and see your score. Try to complete all levels and see how your skills improve!
